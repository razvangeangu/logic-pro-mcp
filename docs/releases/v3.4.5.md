## Summary

v3.4.5 finalizes the source-side Logic Pro 12.2 mixer verification work from Issues #10-#13. The source tag is pushed, but the stable GitHub Release artifacts are not published yet because the release workflow correctly rejects stable tags when notarization secrets are missing.

- #10: host-originated fader writes can verify through independent AX readback when Logic does not emit an MCU echo.
- #11: `logic://mixer` carries provenance and can refresh from AX poll on Logic 12.2 instead of staying stuck on the connect-time MCU value.
- #12: mixer strips expose AX-sourced plugin slot snapshots with `plugins_source:"ax"`; Scripter plugin writes are Honest-Contract-shaped and fail closed on invalid values or unverified track selection.
- #13: `insert_plugin` is restored only as a guarded path: Level-2 confirmation, stock allowlist, occupied-slot refusal, and AX slot readback.
